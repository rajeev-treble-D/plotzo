Installation Guide

1 Prerequisites

Ensure you have Node.js (v18+) and a MySQL server (like XAMPP or MariaDB) running.

2 Deployment

# Start Backend Server

cd puku-crm-backend && npm start

# Start Frontend Panel

cd puku-crm-frontend && npm run dev

3 Auto-Setup

Navigate to http://localhost:3000. The system will automatically redirect you to the Installer. Follow the on-screen steps to configure your database credentials and create your admin account.

Core Modules

Puku CRM is built with a modular architecture, allowing each component to handle specific business logic elegantly.

Dashboard

Centralized hub providing visual summaries of sales, recent activity, and key performance indicators (KPIs).

Customers

Manage your complete client database with detailed profiles, contact history, and relationship status.

Sales

Track every transaction from pending to completed. Generate PDF invoices and monitor revenue trends.

Expenses

Categorize and log business spending. View monthly summaries to maintain financial health.

Projects

End-to-end project tracking. Assign clients, manage budgets, and monitor deadlines effortlessly.

Leads

Capture and nurture potential opportunities. Move leads through stages from "New" to "Won".

Tasks

Action-oriented task management with priorities and due dates. Assigned directly to team members.

Inventory (Stock)

Real-time inventory tracking. Monitor stock levels, SKU management, and unit pricing per category.

Reports

Data-driven reports on finance and operations. AI-powered summaries for strategic decision-making.

Support

Integrated ticketing system for customer queries. Track resolution times and agent performance.

Users & Roles

Granular permissions management. Define roles (Admin, Agent, Manager) and control system access.

Activity Logs

Track every change across the system. Audit who did what and when for maximum accountability.

Settings

Full system customization. Configure appearance, API keys, and business defaults from a single panel.
